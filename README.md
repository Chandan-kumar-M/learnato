Learnato Hackathon - Discussion Forum (MVP)
- frontend: React + Vite + Tailwind (search + mark as answered)
- backend: Node + Express + MongoDB
Quick setup:
1. Ensure Node.js (v16+/v18+) and MongoDB are installed.
2. Backend: cd backend -> copy .env.example to .env -> npm install -> npm start
3. Frontend: cd frontend -> npm install -> npm run dev
